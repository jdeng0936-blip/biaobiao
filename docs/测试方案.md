## 九、缺陷管理

- **严重**（P0）：系统崩溃、数据丢失、安全漏洞 → 立即修复
- **主要**（P1）：核心功能异常、数据错误 → 当天修复
- **次要**（P2）：UI 瑕疵、文案错误 → 排期修复
- **建议**（P3）：体验优化建议 → 评估后排入迭代
